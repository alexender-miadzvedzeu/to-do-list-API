To do list API

How to use this API?

Get all tasks

URL: http://localhost:7777/tasks

METHOD: get

Code example:
        const getAllTasks = async () => {
          const resopnse = await fetch('http://localhost:7777/tasks')
          const tasks = await resopnse.json()
          return tasks;
        }

Post new task

URL: http://localhost:7777/tasks

METHOD: post

Headers: 

        {
          "Content-Type": "application/json"
        }

Code example:
        const postNewTask = async (task) => {
          const resopnse = await fetch('http://localhost:7777/tasks', {
            method: 'POST',
            headers: {
              "Content-Type": "application/json"
            },
            body: JSON.stringify(task)
          })
          const request = await resopnse.json()
          return request;          
        }

Task object should be like this:

        const task = {
          title: "Title text",
          text: "Lorem text...",
          creation_date: "1645966986306", // Date.now()
          task_status: "new"
        }

Note!

All task object fields are required

Change task

URL: http://localhost:7777/tasks

METHOD: put

Headers: 

        {
          "Content-Type": "application/json"
        }

Code example:
        const postNewTask = async (task) => {
          const resopnse = await fetch('http://localhost:7777/tasks', {
            method: 'PUT',
            headers: {
              "Content-Type": "application/json"
            },
            body: JSON.stringify(task)
          })
          const request = await resopnse.json()
          return request;          
        }

Task object should be like this:

        const task = {
          title: "Title text",
          text: "Lorem text...",
          creation_date: "1645966986306", // Date.now()
          task_status: "new"
          id: "...some id from tasks"
        }

Note!

All task object fields are required

In put requiest you should add id in to task object

Delete task

URL: http://localhost:7777/tasks?id=...some_task_id

METHOD: delete

Code example:
        const deleteTask = async (id) => {
          const resopnse = await fetch(`http://localhost:7777/tasks?id=${id}`, {
            method: 'DELETE',
          })
          const request = await resopnse.json()
          return request;
        }
